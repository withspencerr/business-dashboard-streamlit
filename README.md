# Business Analytics Dashboard (Streamlit + Plotly)

**Goal:** Build an interactive dashboard to visualize key business metrics.  
**Tools:** Python · Streamlit · Plotly · Pandas  

## Highlights
- ✅ Interactive filters (Region, Product)
- ✅ Dynamic sales & profit charts
- ✅ Real-time visual feedback for decision-making
- ✅ Clean and executive-ready dashboard layout  

## Demo Preview
- Sales trend line chart  
- Regional sales bar chart  
- Product performance by profit margin  

## How to Reproduce
1. Open `app.py` in **Google Colab** or any IDE  
2. Run:
   ```bash
   streamlit run app.py
